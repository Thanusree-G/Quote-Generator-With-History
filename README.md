# Quote Generator with History

## Overview
This project is a Flask-based web application that fetches random quotes from an external API and stores them in a SQLite database as quote history.

## Features
- Fetch random quotes from API
- Display quotes dynamically
- Store quote history in database
- View previously generated quotes

## Technologies Used
- Python
- Flask
- SQLite
- Requests
- HTML
